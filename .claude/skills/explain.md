用户想要理解一个 AI 推理系统相关的技术概念。

请按照以下结构回答：

1. **一句话定义**：用最简洁的语言说明它是什么。
2. **系统地图定位**：它属于推理系统的哪一层？（模型结构 / Kernel计算 / KV Cache / 并行策略 / 通信 / Runtime引擎 / 调度部署 / 硬件网络）
3. **解决的瓶颈**：它解决了这一层的什么具体问题？
4. **为什么现在重要**：什么变化让这个瓶颈变得突出？
5. **核心原理**：用 2-3 段话解释它的工作机制，假设读者有基本的深度学习和系统知识。
6. **对比**：和同层的其他方案（前代或竞品）相比，关键区别是什么？
7. **一个类比**：用一个日常类比帮助直觉理解。

参考项目中的推理系统地图：docs/inference-system-map.md

用户要解释的概念：$ARGUMENTS
